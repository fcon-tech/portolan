How to use this report

Tabs: Overview (landscape + where to start), Findings (ranked pain by folder/list), Gaps (layers not run), Graph hints (optional portolan map sidecar).

Search in the header queries the local code index — not an LLM.

Findings tab — pick a view (Tour, Code pain, Config), filter by kind/severity, click a row for tool-backed detail and source preview.

Graph hints — populated only when the scan included --with-map-bridge or you ran map + build-map-bridge.sh manually; empty state is honest, not missing findings.
