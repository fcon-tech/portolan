# Apache Bigtop Smoke Fixture

This is a small Apache Bigtop-shaped smoke fixture used by historical
acceptance preflight records. It is intentionally tiny and does not represent a
complete Bigtop checkout.

Use it only when a test or review needs a named Bigtop-shaped local fixture.
For normal CLI tests, prefer neutral fixtures such as `internal/testfixtures/map-command/`
or `internal/testfixtures/landscape-map/`.
