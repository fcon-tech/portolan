# Contract: FCON And Portolan GitHub Pages Site

## Site Contract

The first site version is acceptable only when:

- it has an FCON organization entry point;
- it has a Portolan project page;
- Portolan claims map to `docs/product-claims.md`;
- install, release, demo, GitHub, contribution, and security links are visible;
- publishing source is documented;
- domain and HTTPS state are evidence-labelled;
- no analytics, forms, tracking scripts, credentials, or server-side behavior
  are added without approval.

## Information Architecture

Recommended v1 pages:

```text
/
  FCON: practical AI engineering tools and consulting
/portolan/
  Portolan: local-first evidence maps for agents
/portolan/demo/
  Apache Bigtop demo route and artifact tour
/portolan/release/
  v0.1.0 install and release notes
```

The site may link to repository docs instead of duplicating full content.

## Copy Boundary

Allowed hero line:

```text
Portolan builds local evidence maps so AI agents can answer codebase questions
with fewer unsupported claims.
```

Disallowed without new evidence:

```text
AI SDLC platform
security scanner
observability replacement
service catalog
complete estate understanding
Cursor UI benchmark
enterprise modernization engine
```
