# Quickstart: Agent Acceptance Matrix

1. Read `docs/product-claims.md`.
2. Open `docs/agent/ACCEPTANCE.md`.
3. Select one harness and target cell.
4. Run the acceptance prompt with only Portolan path, target path, and output path.
5. Record lane output under `docs/specs/041-agent-acceptance-matrix/reviews/`.
6. Update product claims only if the lane evidence supports broader wording.
