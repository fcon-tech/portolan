# Design — multiscale-system-drilldown

## Decision

Introduce a multiscale drill-down model (ecosystem → capability → component →
module/package/concept) with evidence backing and honest-empty per scale.

## Status

Recorded intent (spec-level). Scale boundaries, evidence sources per scale, and
the visual interaction are design-TBD. Composes with semantic-investigation
(capability/concept scales) and overlap findings (capability scale).

## Reversibility

High. Additive navigation model; each scale degrades to honest-empty.
